# 📊 Customer Churn Analysis & Retention Dashboard

An end-to-end Customer Churn Analysis project developed using **Python and Power BI** to identify customer churn drivers, analyze retention patterns, and support data-driven business decisions.

---

# 📌 Project Overview

An interactive dashboard was built to analyze telecom customer behavior, identify factors influencing churn, and provide actionable insights for improving customer retention strategies.

The project answers questions such as:

- Which customers are more likely to churn?
- How does contract type influence churn?
- Does online security affect customer retention?
- How do payment methods and demographics impact churn?
- Which customer segments require targeted retention strategies?

# 🔍 Key Findings

- Approximately **26.58%** of customers have churned.
- Customers with **month-to-month contracts** exhibit the highest churn.
- Customers without **online security services** are more likely to churn.
- Churned customers pay **higher average monthly charges**.
- Customers with **shorter tenure** demonstrate higher churn tendencies.
- Customers without **partners or dependents** are more prone to churn.
