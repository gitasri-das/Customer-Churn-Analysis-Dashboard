Customer Churn Analysis & Retention Dashboard
📌 Project Overview

This project analyzes customer churn behavior in a telecom company and identifies the factors influencing customer retention and churn. The project combines Python-based data preprocessing and exploratory data analysis with an interactive Power BI dashboard for business insights.

The dashboard helps answer questions such as:

Which customers are more likely to churn?
How do contract types influence churn?
Does online security impact customer retention?
How do payment methods and demographics affect churn?

Key Business Insights
📈 Key Findings
Approximately 26.58% customers have churned.
Customers with month-to-month contracts exhibit the highest churn.
Customers without online security are significantly more likely to churn.
Churned customers pay higher average monthly charges.
Customers with shorter tenure demonstrate higher churn tendencies.
Customers without partners or dependents are more prone to churn.
